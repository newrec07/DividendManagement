\## 1. 목표

\- 배당성장주 투자의 본질인 \*\*"싸게 사는 것"\*\* 에 가장 높은 가중치 집중 (절대 가치 35점)

\- 9요소 정밀형과 병행 사용 가능. 두 버전 점수가 모두 80점 이상일 때 신뢰도 최고

\## 3. 요소별 판별 로직  
<br>
\*\*F1 — 역대 배당수익률 위치\*\*  
\`\`\`  
\# 밴드 미형성(분모 = 0) 처리 — 배당 시작 초기 종목, 5년간 배당률 변동 없는 종목  
\# ※ 심플형은 F1 배점이 35점으로 201보다 크므로 밴드 미형성 시 UI 경고 더 명확히 표시  
IF (Max\_Yield\_5Y - Min\_Yield\_5Y) == 0:  
    Score\_F1 = 0  # 판단 불가, UI에 '밴드 데이터 부족 — 201 정밀형으로 재검토 권고' 표시  
ELSE:  
    Score\_F1 = MIN(35, MAX(0,  
        ((Current\_Yield - Min\_Yield\_5Y) / (Max\_Yield\_5Y - Min\_Yield\_5Y)) \* 35  
    ))  
\`\`\`  
<br>
\- 5년 최고를 초과하면 상한값 처리 → 만점 35점  
<br>
\*\*F2 — 이익 성장 & 배당성향 종합\*\*  
\`\`\`  
\# 배당성장률 기준: 203 문서 Weighted\_DGR과 동일한 5년 선형가중평균 DGR 사용  
\# Weighted\_DGR = (DGR\_Y4×1 + DGR\_Y3×2 + DGR\_Y2×3 + DGR\_Y1×4 + DGR\_Y0×5) / 15  
<br>
IF Payout\_Ratio <= 0.65 AND Weighted\_DGR >= 0.10 → 20점  
IF Payout\_Ratio <= 0.75 AND Weighted\_DGR > 0.05 → 10점  
ELSE → 0점  
<br>
\# ※ 한계 인지: AND 조건으로 인해 아래 케이스에서 False Negative 발생 가능  
#    배당성향 60%(안전) + DGR 8%(준수) → 0점  
#    배당성향 74%(경계) + DGR 6%(낮음) → 10점  
#    경계값 근처 종목은 반드시 201 정밀형으로 재검토  
\`\`\`  
<br>
\*\*F3 — MDD\*\*  
\`\`\`  
MDD = (Current\_Price - High\_52W) / High\_52W  
IF MDD <= -0.20 → 15점  
IF MDD <= -0.15 → 8점  
ELSE → 0점  
\`\`\`  
<br>
\*\*F5 — 13주/40주 이동평균선\*\*  
\`\`\`  
IF SMA\_13W < SMA\_40W → 8점  
IF SMA\_13W >= SMA\_40W → 0점  
\`\`\`  
<br>
\*\*F6 — 14주 RSI\*\*  
\`\`\`  
IF RSI\_14W <= 30 → 10점  
IF RSI\_14W <= 40 → 7점  
IF RSI\_14W <= 50 → 3점  
ELSE → 0점  
※ 반드시 F7 볼린저밴드와 함께 해석, 단독 판단 금지  
\`\`\`  
<br>
\*\*F7 — 40주 볼린저밴드 위치\*\*  
\`\`\`  
BB\_Mid = SMA\_40W  
BB\_Upper = SMA\_40W + (STD\_40W × 2)  
BB\_Lower = SMA\_40W - (STD\_40W × 2)  
<br>
IF Current\_Price <= BB\_Lower → 12점  
IF Current\_Price <= BB\_Lower + (BB\_Mid - BB\_Lower) × 0.33 → 8점  
IF Current\_Price < BB\_Mid → 4점  
IF Current\_Price >= BB\_Mid → 0점  
<br>
최강 매수 신호: RSI\_14W <= 30 AND Current\_Price <= BB\_Lower  
\`\`\`
