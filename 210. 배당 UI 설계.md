## 210\. 배당 UI 설계

\# 화면 설계 v1.1  
<br>
\---  
<br>
\## 전체 화면 구조  
\`\`\`  
로그인/회원가입  
↓  
대시보드 (메인)  
├─ 종목 상세  
│   ├─ 시뮬레이션  
│   └─ 뉴스 (종목 상세 내 블록)  
└─ 포트폴리오  
    └─ 배당 수령 기록  
\`\`\`  
<br>
\---  
<br>
\## Screen 0 — 로그인 / 회원가입  
<br>
\*\*기능\*\*  
\- 이메일 + 비밀번호 입력  
\- 로그인 버튼 → JWT 토큰 발급 → localStorage 저장  
\- 회원가입 버튼  
\- 에러 메시지 (이메일 중복, 비밀번호 오류 등)  
\- 미로그인 상태로 다른 화면 접근 시 자동 리다이렉트  
\- 토큰 만료 시 자동 로그아웃 → 로그인 화면으로 이동  
<br>
\---  
<br>
\## Screen 1 — 대시보드 (메인)  
<br>
\*\*핵심 기능\*\*  
\- 전 종목 점수 테이블 (201 / 202 / 203 / Red Flag)  
\- 점수 기준 정렬  
\- 필터: 전체 / Strong Buy / Watch / 보유중 / C2\_BLOCKED  
\- 마지막 갱신일 표시  
\- 종목 추가 버튼 (티커 입력 → On-demand 수집)  
\- 종목 클릭 → 종목 상세로 이동  
\- 상단 수동 갱신 버튼  
\- 상단 로그아웃 버튼  
<br>
\*\*테이블 컬럼\*\*  
| 티커 | 종목명 | 201점수 | 202점수 | 203등급 | C2 | RedFlag | YellowFlag | 갱신일 |  
|------|--------|--------|--------|--------|----|--------|-----------|--------|  
<br>
\*\*배지 및 색상 기준\*\*  
\`\`\`  
201 신호:  
  STRONG\_BUY  → 🟢 초록  
  WATCH       → 🟡 노랑  
  HOLD        → ⚪ 회색  
  C2\_BLOCKED  → 🟠 주황 + ⚠️ 배지  
<br>
203 등급:  
  STRONG  → 🟢 초록  
  CAUTION → 🟡 노랑  
  DANGER  → 🔴 빨강  
  미계산  → ⚪ 회색  
<br>
Red Flag → 🔴 숫자 배지  
Yellow Flag → 🟡 숫자 배지  
\`\`\`  
<br>

\## Screen 2 — 종목 상세  
<br>
\*\*핵심 기능\*\*  
<br>
블록 A — 현재 시장 상태  
\`\`\`  
현재가 / 배당수익률 / PER  
MDD (52주 고점 대비 하락률)  
VIX 현재값  
\`\`\`  
<br>
블록 B — 201 정밀형 점수  
\`\`\`  
총점 + 신호 배지 (STRONG\_BUY / WATCH / HOLD / C2\_BLOCKED)  
카테고리별 점수: C1(30) / C2(30) / C3(20) / C4(20)  
요소별 점수 + 판단 근거 한 줄 텍스트  
F1/F2 밴드 미형성 시 → "밴드 데이터 부족" 표시  
C2\_BLOCKED 시 → "⚠️ C2 이익 체력 0점 — 매수 보류" 경고 배너  
\`\`\`  
<br>
블록 C — 202 심플형 점수  
\`\`\`  
총점 + 신호 배지 (IMMEDIATE\_BUY / WATCH / WAIT)  
카테고리별 점수: C1(35) / C2(20) / C3(15) / C4(30)  
요소별 점수 + 판단 근거 한 줄 텍스트  
F1 밴드 미형성 시 → "밴드 데이터 부족 — 201 정밀형으로 재검토 권고" 표시  
\`\`\`  
<br>
블록 D — 203 지속가능성  
\`\`\`  
LAYER1(70) / LAYER2(30) 점수  
등급 배지 (STRONG / CAUTION / DANGER)  
Yellow Flag 목록 (있을 경우)  
Red Flag 목록 (있을 경우)  
Red Flag 이력 (해제된 경우) → "이전 Red Flag 이력 있음 (해제일: YYYY-MM-DD)"  
FCF Yellow Flag 시 → "⚠️ FCF 커버리지 1년 미달 — 모니터링 강화" 표시  
\`\`\`  
<br>
블록 E — 뉴스  
\`\`\`  
Alpha Vantage 뉴스 카드 목록  
  - 감성 배지 (Bullish / Neutral / Bearish)  
  - 기사 제목  
  - 원문 링크 → 클릭 시 새 탭으로 이동 (누락 금지)  
  - 발행일시  
  - "뉴스 감성 기반 근사값" 안내 문구 표시  
\`\`\`  
<br>
\*\*하단 액션 버튼\*\*  
\`\`\`  
\[시뮬레이션 보기\]  
  활성 조건: grade\_203 = 'STRONG' AND red\_flag\_count = 0  
  Red Flag 이력 있으면 활성이나 이력 경고 툴팁 표시  
  그 외 → 비활성 + 비활성 사유 툴팁  
<br>
\[매수 기록하기\] → 항상 활성 (점수 무관)  
\`\`\`

\## Screen 3 — 시뮬레이션  
<br>
\*\*입력값\*\*  
\`\`\`  
매수가 입력 (기본값: 현재가 자동)  
수량 입력  
시나리오 탭 (Base / Bull / Bear)  
\`\`\`  
<br>
\*\*상단 안내 문구\*\*  
\`\`\`  
"단기 구간(Q+1~Q+4)은 애널리스트 컨센서스 기반으로 고정됩니다.  
 시나리오 변경은 중기(Q+5~) 이후 구간에 반영됩니다."  
\`\`\`  
<br>
\*\*출력 #1 — 분기별 예상 DPS 테이블\*\*  
\`\`\`  
컬럼: 분기 | DPS Base | DPS Bull | DPS Bear | 신뢰도 | 데이터 소스  
색상:  
  초록 = 단기 Q+1~Q+4 (컨센서스 고정, 시나리오 전환 시 불변)  
  노랑 = 중기 Q+5~Q+12 (블렌딩)  
  회색 = 장기 Q+13~Q+40 (DGR 리니어)  
\`\`\`  
<br>
\*\*출력 #2 — 누적 배당 곡선 (Recharts)\*\*  
\`\`\`  
X축: 분기 (Q+1 ~ Q+40)  
Y축: 누적 배당금 총액  
3개 라인: Bull / Base / Bear  
원금 회수 시점 수직선 표시  
\`\`\`  
<br>
\*\*출력 #3 — YOC 추적 차트 (Recharts)\*\*  
\`\`\`  
X축: 연도 (Year 1 ~ Year 10)  
Y축: YOC (%)  
현재 배당수익률 기준선  
사용자 설정 목표 YOC 선  
\`\`\`  
<br>
\*\*출력 #4 — 목표 매수가 역산\*\*  
<br>
\`\`\`  
입력: 원하는 목표 YOC (%)  
출력: 목표 YOC 달성을 위한 최대 매수가  
계산: 목표가 = 예상 연간 DPS(Y+1) / 목표 YOC  
\`\`\`  
<br>
\*\*출력 #5 — 신호 요약 배너\*\*  
\`\`\`  
\[컨센서스 방향\] 상향 / 유지 / 하향  
\[뉴스 톤\] 긍정 / 중립 / 부정  
\[충돌 감지\] 없음 / ⚠️ DIVERGENCE ALERT  
\[시뮬레이션 신뢰도\] HIGH / MEDIUM / LOW  
\`\`\`  
<br>
\*\*시나리오 전환 동작\*\*  
\`\`\`  
Base / Bull / Bear 탭 클릭 시:  
  단기(Q+1~Q+4) → 고정. 변경 없음.  
  중기/장기 → 프론트에서 DGR값만 변경하여 즉시 재계산  
  서버 재호출 없음  
\`\`\`

\## Screen 4 — 포트폴리오  
<br>
\*\*핵심 기능\*\*  
<br>
블록 A — 전체 요약  
\`\`\`  
총 투자금액  
현재 평가금액  
미실현 손익 (%)  
연간 예상 배당금  
포트폴리오 YOC  
\`\`\`  
<br>
블록 B — 종목별 보유 현황  
\`\`\`  
티커 / 보유수량 / 평균단가  
현재가 / 수익률  
예상 연간 배당금 / YOC  
Red Flag 보유 종목 → 🔴 경고 아이콘 + "즉시 재검증 필요" 툴팁  
Yellow Flag 보유 종목 → 🟡 경고 아이콘 + "모니터링 강화" 툴팁  
\`\`\`  
<br>
블록 C — 배당 수령 이력  
\`\`\`  
월별 수령 배당금 바차트 (Recharts)  
종목별 배당 수령 내역 테이블  
\`\`\`  
<br>
\*\*액션 버튼\*\*  
\`\`\`  
\[매수 기록\]  
  → 종목 / 날짜 / 수량 / 단가 입력 폼  
<br>
\[매도 기록\]  
  → 동일 구조  
<br>
\[배당 수령 기록\]  
  → 종목 / 날짜 / DPS / 수량 입력 폼  
\`\`\`

\## 공통 요소  
<br>
\*\*하단 탭 네비게이션\*\*  
\`\`\`  
대시보드 | 포트폴리오  
\`\`\`  
<br>
\*\*상단 바\*\*  
\`\`\`  
좌측: 앱 로고  
우측: 수동 갱신 버튼 / 로그아웃 버튼  
\`\`\`  
<br>
\*\*전역 경고 배너 (고정)\*\*  
\`\`\`  
Red Flag 발생 시:  
  → 전 화면 상단에 🔴 경고 배너 고정  
  → "Red Flag {n}개 감지 — {종목명} 즉시 재검증 필요"  
  → 배너 클릭 시 해당 종목 상세로 이동  
<br>
데이터 갱신 실패 시:  
  → "데이터 갱신 실패 — 이전 데이터 표시 중" 배너  
\`\`\`  
<br>
\*\*일부 지표 계산 불가 표시\*\*  
\`\`\`  
F1/F2 밴드 미형성, 데이터 누락 등  
→ 해당 요소에 "계산 불가" 표시  
→ 총점 옆에 "일부 지표 제외됨" 안내  
\`\`\`
