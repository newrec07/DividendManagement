\## 2. 핵심 개념 정의  
<br>
\*\*① YOC (Yield on Cost)\*\*  
\> 매수 당시 가격 기준으로 미래에 받게 될 배당수익률.  
\> DGR이 복리로 누적되면서 YOC가 상승하는 구조가 배당성장주 투자의 본질.  
\`\`\`  
YOC\_t = 예상 연간 DPS\_t / 최초 매수가  
\`\`\`  
<br>
\*\*② DGR (Dividend Growth Rate)\*\*  
\> 배당금의 전년 동기 대비 성장률.  
\> 단순 평균이 아닌 최근일수록 가중치를 높인 선형가중평균 사용.  
\> 202, 203 문서의 Weighted\_DGR과 동일한 값 참조 (계산 중복 제거)  
<br>
\*\*③ Forward DPS 예측 체인\*\*  
\> 실제 발표 → 컨센서스 EPS × Payout Ratio → DGR 리니어  
\> 구간이 멀어질수록 신뢰도가 낮아지므로 데이터 소스를 단계적으로 전환.

\## 4. DGR 기준값 산출 로직  
<br>
\### 4-1. 특별배당 제거  
\`\`\`  
\# 아래 3개 조건 중 하나라도 해당 시 특별배당으로 분류, 제외  
조건 1: DPS\_Q > 직전 4분기 평균 DPS × 3.0  (직전 분기 단독 비교보다 안정적)  
조건 2: FMP dividend 응답의 label 필드에 "special" 키워드 포함  
조건 3: DPS\_Q > DPS\_Q-1 × 3.0 AND DPS\_Q-1 > 0  (직전 분기 0이면 조건 미적용)  
<br>
→ 정규화된 DPS 시계열 확정  
\`\`\`  
<br>
\### 4-2. 연간 DGR 산출 (분기 기준)  
\`\`\`  
DGR\_Y = (DPS\_정규화\_최근4Q 합산 / DPS\_정규화\_전년4Q 합산) - 1  
→ 5개년 DGR\_Y0 ~ DGR\_Y4 산출  
\`\`\`  
<br>
\### 4-3. 선형 가중평균 (203 문서 계승)  
\`\`\`  
Weighted\_DGR = (DGR\_Y4×1 + DGR\_Y3×2 + DGR\_Y2×3 + DGR\_Y1×4 + DGR\_Y0×5) / 15  
\`\`\`  
<br>
\### 4-4. 시나리오별 DGR 확정  
\`\`\`  
Base  = Weighted\_DGR  
Bull  = Weighted\_DGR + 2%p  (실적 개선, 인상 가속)  
Bear  = max(Weighted\_DGR - 3%p, -5%)  (성장 둔화 or 1회 동결. 최대 -5%까지 허용)  
Freeze = 0%  (Red Flag 발동 시 강제 적용)  
\`\`\`

\## 5. 구간별 Forward DPS 예측 로직  
<br>
\### 구간 정의  
\`\`\`  
단기: Q+1 ~ Q+4   (향후 1년)      신뢰도 HIGH  
중기: Q+5 ~ Q+12  (1년 ~ 3년)    신뢰도 MEDIUM  
장기: Q+13 ~ Q+40 (3년 ~ 10년)   신뢰도 LOW (리니어)  
\`\`\`  
<br>
\### 단기 로직 (Q+1 ~ Q+4)  
\`\`\`  
우선순위 1: 실제 공시 배당금 (이미 선언된 경우)  
우선순위 2: Forward EPS 컨센서스 × 과거 5년 평균 Payout Ratio  
우선순위 3: 직전 분기 DPS × (1 + Weighted\_DGR / 4)  
<br>
신뢰도 조정:  
IF EPS 컨센서스 분산 > 20% → 신뢰도 밴드 ±15% 확장  
IF EPS 리비전 하향 중 → Bear Case 가중치 상향  
<br>
\# ※ 시나리오(Base/Bull/Bear) 전환 시 단기 구간은 변경 없음  
#    단기는 컨센서스/공시 기반 고정값. DGR 변경 영향 없음.  
#    UI에 "단기 구간(Q+1~Q+4)은 컨센서스 기반으로 고정됩니다" 안내 문구 표시  
\`\`\`  
<br>
\### 중기 로직 (Q+5 ~ Q+12)  
\`\`\`  
컨센서스 커버리지가 있는 분기:  
  DPS\_predicted = EPS\_fwd × Payout\_avg  
  단, 컨센서스 신뢰도 가중치 반영  
<br>
컨센서스 없는 분기:  
  DPS\_predicted = DPS\_직전분기 × (1 + Weighted\_DGR / 4)  
<br>
블렌딩:  
  최종 DPS = 컨센서스 기반 × w + DGR 기반 × (1-w)  
<br>
  # 블렌딩 가중치 w 정규화 공식  
  eps\_spread = (eps\_high - eps\_low) / eps\_consensus  # 상대 분산  
  w = max(0, min(1, 1 - eps\_spread))  
  # spread 0% (만장일치) → w=1 (컨센서스 100% 반영)  
  # spread 100% 이상 → w=0 (DGR 100% 반영)  
\`\`\`  
<br>
\### 장기 로직 (Q+13 ~ Q+40)  
<br>
\`\`\`  
DPS\_Q+n = DPS\_Q+12 × (1 + Weighted\_DGR / 4)^(n-12)  
<br>
시나리오별 독립 계산:  
  Base:  Weighted\_DGR  
  Bull:  Weighted\_DGR + 2%p  
  Bear:  max(Weighted\_DGR - 3%p, -5%)  
<br>
※ 선형 복리 적용. 모델 복잡도 불필요.  
\`\`\`

\## 6. 뉴스 vs 컨센서스 충돌 감지 로직  
\`\`\`  
STEP 1. 컨센서스 방향 판별  
  IF EPS 리비전 상향 → CONSENSUS = POSITIVE  
  IF EPS 리비전 유지 → CONSENSUS = NEUTRAL  
  IF EPS 리비전 하향 → CONSENSUS = NEGATIVE  
<br>
STEP 2. 뉴스 톤 판별 (203 L2-F3 키워드 재활용)  
  긍정 키워드 수 vs 부정 키워드 수 비교  
  → NEWS\_TONE = POSITIVE / NEUTRAL / NEGATIVE  
<br>
STEP 3. 충돌 매트릭스  
  CONSENSUS     NEWS\_TONE      결과  
  POSITIVE   ×  POSITIVE   =  신뢰도 유지  
  POSITIVE   ×  NEGATIVE   =  ⚠️ DIVERGENCE ALERT → 밴드 ±20% 확장  
  NEGATIVE   ×  POSITIVE   =  ℹ️ 회복 가능성 모니터링  
  NEGATIVE   ×  NEGATIVE   =  🔴 Bear Case 강제 활성화 + Freeze 검토  
<br>
STEP 4. Divergence Alert 발동 시  
  → 해당 구간 예측 밴드 ±20% 추가 확장  
  → 화면 상단에 경고 배너 표시  
\`\`\`

\## 7. 출력 항목 정의  
<br>
\### 출력 #1 — 분기별 예상 DPS 테이블  
\`\`\`  
컬럼: 분기 | DPS Base | DPS Bull | DPS Bear | 신뢰도 | 데이터 소스  
색상 구분:  
  초록 = 단기 (실제 발표/컨센서스 기반, 시나리오 전환 시 고정)  
  노랑 = 중기 (컨센서스 블렌딩)  
  회색 = 장기 (DGR 리니어)  
\`\`\`  
<br>
\### 출력 #2 — 누적 수령 배당금 곡선  
\`\`\`  
X축: 분기 (Q+1 ~ Q+40)  
Y축: 누적 배당금 총액 (매수가 기준)  
3개 라인: Bull / Base / Bear  
표시: 원금 회수 시점 (누적 배당 = 매수가)  
\`\`\`  
<br>
\### 출력 #3 — YOC 추적 차트  
\`\`\`  
X축: 연도 (Year 1 ~ Year 10)  
Y축: YOC (%)  
기준선: 현재 배당수익률 (매수 시점)  
목표선: 사용자 설정 목표 YOC  
\`\`\`  
<br>
\### 출력 #4 — 목표 매수가 역산  
\`\`\`  
입력: 원하는 목표 YOC (예: 5%)  
출력: 목표 YOC 달성을 위한 최대 매수가  
계산: 목표가 = 예상 연간 DPS(Y+1) / 목표 YOC  
\`\`\`  
<br>
\### 출력 #5 — 신호 요약 배너  
\`\`\`  
\[컨센서스 방향\] EPS 상향 조정 중  
\[뉴스 톤\] 중립  
\[충돌 감지\] 없음  
\[시뮬레이션 신뢰도\] HIGH  
\`\`\`
