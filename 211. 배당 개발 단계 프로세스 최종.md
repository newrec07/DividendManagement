\*\*작업 순서\*\*

<br>
2\. financials.py  
   → FMP 5년치 수집

   → Weekly Refresh용  
<br>
6\. news.py  
   → Alpha Vantage NEWS\_SENTIMENT 연동  
   → url 필드 반드시 포함 확인

\## Phase 3 — 점수 엔진  
<br>
\*\*작업 내용\*\*  
\`\`\`  
engines/  
├── score\_201.py    정밀형 F1~F10 계산  
├── score\_202.py    심플형 F1~F7 계산  
├── score\_203.py    L1/L2 계산 + Flag 카운팅  
└── score\_runner.py 3개 엔진 통합 실행 + DB 저장  
\`\`\`  
<br>
\*\*작업 순서\*\*  
\`\`\`  
1\. score\_201.py  
   → 요소별 함수 단위로 분리 작성  
   → F1/F2 분모 0 처리 (밴드 미형성 → 0점 + null 플래그)  
   → F3 기준: quarterly\_financials 최근 확정 발표 분기 고정  
   → F4 FCF CAGR: TTM(최근 4분기 합산) 연간 기준  
   → C2 필터: F3=0 OR F4=0 시 signal\_201 = 'C2\_BLOCKED'  
   → c2\_blocked = TRUE 저장  
<br>
2\. score\_202.py  
   → F1 분모 0 처리 (35점 배점, UI 경고 강화)  
   → F2 배당성장률: Weighted\_DGR 5년 선형가중평균 사용  
   → 신호 3단계: IMMEDIATE\_BUY / WATCH / WAIT  
<br>
3\. score\_203.py  
   → L1-F1 FCF 커버리지: 1년 Fail → Yellow Flag / 2년 연속 → Red Flag  
   → L1-F6 부채비율: tickers.json의 sector\_de\_ratio 참조  
   → L1-F8 ROIC vs WACC: tickers.json의 wacc 참조  
   → L2-F3 뉴스 감성: Alpha Vantage 기반 근사값  
   → Red Flag 카운팅 + yellow\_flag\_count 별도 집계  
   → red\_flag\_history 이력 관리  
<br>
4\. score\_runner.py  
   → 3개 엔진 순서대로 실행  
   → 결과 score\_cache\_global에 upsert (공용 테이블)  
   → 소스 매핑 주석 필수:  
      현재값 → snapshot.json  
      과거 시계열 → price\_history.json / financials.json  
\`\`\`  
<br>
\*\*완료 기준\*\*  
\`\`\`  
□ 실제 종목 3개 이상 점수 계산 완료  
□ 201/202 점수 수동 계산값과 대조 확인  
□ F1/F2 분모 0 처리 동작 확인  
□ F4 TTM 연산 정상 동작 확인  
□ C2\_BLOCKED 발동 케이스 테스트 확인  
□ Red Flag 1년/2년 구분 동작 확인  
□ Yellow Flag 카운팅 확인  
□ score\_cache\_global DB 저장 확인

<br>
\*\*작업 순서\*\*  
\`\`\`  
1\. 공통 레이아웃  
   └─ 하단 탭 네비게이션 (대시보드 / 포트폴리오)  
   └─ 상단 바 (로그아웃 / 수동갱신)  
   └─ Red Flag 전역 배너 컴포넌트  
   └─ 데이터 갱신 실패 배너 컴포넌트  
<br>
2\. 로그인 화면 (Screen 0)  
   └─ JWT 토큰 localStorage 저장  
   └─ 미인증 접근 시 리다이렉트  
<br>
3\. 대시보드 (Screen 1)  
   └─ TanStack Table 종목 테이블  
   └─ 필터 버튼 (C2\_BLOCKED 포함)  
   └─ 배지 색상 구분  
   └─ 종목 추가 입력  
<br>
4\. 종목 상세 (Screen 2)  
   └─ 블록 A~E 순서대로  
   └─ C2\_BLOCKED 경고 배너  
   └─ Yellow Flag / Red Flag 이력 표시  
   └─ 뉴스 카드 + 원문 링크 (새 탭)  
   └─ 시뮬레이션 버튼 활성/비활성 조건  
<br>
5\. 시뮬레이션 (Screen 3)  
   └─ 단기 구간 고정 안내 문구  
   └─ Recharts 차트 2개  
   └─ 시나리오 탭 전환 (단기 고정 / 중기·장기 즉시 재계산)  
   └─ 신호 요약 배너  
<br>
6\. 포트폴리오 (Screen 4)  
   └─ 요약 블록  
   └─ 보유 현황 테이블 (Red Flag / Yellow Flag 아이콘)  
   └─ 월별 배당 바차트  
   └─ 매수/매도/배당 입력 폼
