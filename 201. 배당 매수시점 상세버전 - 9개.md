## 201\. 배당 매수시점 상세버전 - 9개

\# 매수 시점 판단 로직 — 정밀형 (9요소 v2.1)  
<br>
\---  
<br>
\## 1. 목표  
펀더멘털이 검증된 배당성장주가 시장의 일시적 공포 또는 과도한 조정으로 주가가 하락했을 때, 감정을 배제하고 기계적으로 매수 타이밍을 포착한다.  
<br>
\- 단순히 주가가 싸졌는지가 아니라, \*\*이익 체력이 살아있는 상태에서 가격만 빠졌는지\*\*를 정밀하게 검증  
\- 4개 카테고리 10개 지표를 100점 만점으로 계량화하여 매수 신호를 객관적으로 판정  
\- 주봉(weekly) 기준 매주 금요일 1회 업데이트 → 일봉 노이즈 제거, 감정 개입 최소화

\## 2. 다루는 요소 정의  
<br>
\### C1. 가치 평가 (30점)  
"지금 이 주가가 역사적으로 싼 구간인가?"  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| F1 | 역대 배당수익률 위치 | 현재 배당률이 과거 5년 최고/최저 밴드 내에서 어느 위치에 있는지. 배당률이 높을수록 주가가 상대적으로 저평가된 상태. 밴드 미형성 시 0점 처리 |  
| F2 | PE 밴드 위치 | 현재 PER이 과거 5년 PER 밴드 내 하단에 가까울수록 이익 대비 주가가 싸진 상태. 배당률 단독 판단 시 발생하는 배당 함정(Dividend Trap) 보완. 밴드 미형성 시 0점 처리 |  
<br>
\### C2. 기업 퍼포먼스 (30점)  
"배당을 지급할 본업 체력이 살아있는가?"  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| F3 | 매출 & EPS 성장 (YoY) | 가장 최근 확정 발표된 분기 기준(FMP/yfinance quarterly\_financials 최근 행). 해당 분기 매출과 EPS가 전년 동기 대비 플러스 성장 중인지. 이익 없는 배당률 상승은 배당 함정 |  
| F4 | 잉여현금흐름(FCF) 추이 | 최근 3년 FCF CAGR. \*\*TTM(최근 4분기 합산) 기준 연간 FCF\*\* 3개년 CAGR로 계산. 분기 단독 값 사용 금지 (계절성 왜곡 방지). 배당금은 장부 이익이 아닌 실제 현금에서 지급되므로 현금 창출력 우상향이 핵심 |  
| F5 | 배당성향 (Payout Ratio) | 순이익 대비 배당금 비율. 60% 이하면 위기 시에도 배당 유지 가능한 안전마진 확보 상태 |  
<br>
\### C3. 시장 심리 (20점)  
"지금 시장이 과도한 공포 상태인가?"  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| F6 | MDD (52주 고점 대비 하락률) | 전고점 대비 현재 주가 하락 폭. 개별 기업의 이익이 훼손되지 않았는데 주가만 빠진 '바겐세일 구간' 판별 |  
| F7 | VIX 지수 | 시장 전체의 공포 온도계. 거시 이슈로 시장 전체가 패닉 상태일 때 우량 배당주도 함께 저평가되는 시점 포착. ※ VIX는 종목 무관 시장 지표로, C3 20점 중 절반이 개별 종목 펀더멘털과 무관하게 적용됨을 인지할 것 |  
<br>
\### C4. 매수 타이밍 — Tawcan 주봉 방식 (20점)  
"떨어지는 칼날인가, 바닥을 다지는 구간인가?"  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| F8 | 13주/40주 이동평균선 크로스오버 | 단기(13주) 이평선이 장기(40주) 이평선 아래에 있으면 하락 추세 = 매수 검토 구간. 13주선이 40주선 위로 크로스하면 추세 반전 확인 후 보유 전환 |  
| F9 | 14주 RSI | 주봉 기준 14주 상대강도지수. 과매도 구간(30 이하) 판별. 반드시 F10 볼린저밴드와 함께 해석, 단독 판단 금지 |  
| F10 | 40주 볼린저밴드 위치 | 40주 이평선 기준 ±2 표준편차 밴드. 하단 터치는 전체 기간의 약 5%만 발생하는 극단적 과매도 구간 |

\## 3. 요소별 판별 로직  
<br>
\*\*F1 — 역대 배당수익률 위치\*\*  
\`\`\`  
\# 밴드 미형성(분모 = 0) 처리 — 배당 시작 초기 종목, 5년간 배당률 변동 없는 종목  
IF (Max\_Yield\_5Y - Min\_Yield\_5Y) == 0:  
    Score\_F1 = 0  # 판단 불가, UI에 '밴드 데이터 부족' 표시  
ELSE:  
    Score\_F1 = MIN(15, MAX(0,  
        ((Current\_Yield - Min\_Yield\_5Y) / (Max\_Yield\_5Y - Min\_Yield\_5Y)) \* 15  
    ))  
\`\`\`  
<br>
\- 현재 배당률이 5년 최저에 가까울수록 0점, 5년 최고에 가까울수록 15점  
\- 5년 최고를 초과하면 상한값 처리 → 만점 15점  
<br>
\*\*F2 — PE 밴드 위치\*\*  
\`\`\`  
\# 밴드 미형성(분모 = 0) 처리 — PER 변동 거의 없는 방어주, 신규 편입 종목  
IF (Max\_PE\_5Y - Min\_PE\_5Y) == 0:  
    Score\_F2 = 0  # 판단 불가, UI에 '밴드 데이터 부족' 표시  
ELSE:  
    Score\_F2 = MIN(15, MAX(0,  
        ((Max\_PE\_5Y - Current\_PE) / (Max\_PE\_5Y - Min\_PE\_5Y)) \* 15  
    ))  
\`\`\`  
\- PE는 낮을수록 좋으므로 분자를 Max\_PE에서 Current\_PE를 뺀 값으로 설정  
<br>
\*\*F3 — 매출 & EPS 성장\*\*  
\`\`\`  
\# 기준: 가장 최근 확정 발표된 분기 (quarterly\_financials 최근 행 고정)  
\# 실적 발표 지연이 있어도 "확정 발표 기준"으로 통일  
IF Rev\_YoY > 0 AND EPS\_YoY > 0 → 10점  
IF Rev\_YoY > 0 OR EPS\_YoY > 0 → 5점  
ELSE → 0점  
\`\`\`  
<br>
\*\*F4 — 잉여현금흐름(FCF) 추이\*\*  
\`\`\`  
\# FCF\_3Y\_CAGR = TTM 연간 FCF 기준 3개년 CAGR  
\# TTM = 최근 4분기 합산. 분기 단독값 사용 금지 (계절성 왜곡 방지)  
\# 계산: (TTM\_FCF\_현재 / TTM\_FCF\_3년전) ^ (1/3) - 1  
<br>
IF FCF\_3Y\_CAGR >= 0.10 → 10점  
IF FCF\_3Y\_CAGR > 0 → 7점  
ELSE → 0점  
\`\`\`  
<br>
\*\*F5 — 배당성향\*\*  
\`\`\`  
IF Payout\_Ratio <= 0.60 → 10점  
IF Payout\_Ratio <= 0.75 → 5점  
ELSE → 0점  
\`\`\`  
<br>
\*\*F6 — MDD\*\*  
\`\`\`  
MDD = (Current\_Price - High\_52W) / High\_52W  
IF MDD <= -0.20 → 10점  
IF MDD <= -0.15 → 7점  
IF MDD <= -0.10 → 4점  
ELSE → 0점  
\`\`\`  
<br>
\*\*F7 — VIX 지수\*\*  
\`\`\`  
\# ※ 설계 인지 사항: VIX는 종목 무관 시장 지표.  
#    시장 공포기에는 모든 종목이 동일하게 최대 10점 보너스를 받음.  
#    F6(MDD)과 함께 해석하여 "시장 공포 + 개별 주가 하락"이 동시에 발생한 경우를  
#    가장 이상적인 매수 구간으로 판단하는 것이 설계 의도.  
<br>
IF VIX\_Current >= 30 → 10점  
IF VIX\_Current >= 25 → 7점  
IF VIX\_Current >= 20 → 4점  
ELSE → 0점  
\`\`\`  
<br>
\*\*F8 — 13주/40주 이동평균선\*\*  
\`\`\`  
IF SMA\_13W < SMA\_40W → 6점  
IF SMA\_13W >= SMA\_40W → 0점  
추가: 13주선이 40주선 위로 크로스 시 → 추가 매수 종료, 보유 전환  
\`\`\`  
<br>
\*\*F9 — 14주 RSI\*\*  
\`\`\`  
IF RSI\_14W <= 30 → 6점  
IF RSI\_14W <= 40 → 4점  
IF RSI\_14W <= 50 → 2점  
ELSE → 0점  
※ 반드시 F10 볼린저밴드와 함께 해석, 단독 판단 금지  
\`\`\`  
<br>
\*\*F10 — 40주 볼린저밴드 위치\*\*  
\`\`\`  
BB\_Mid = SMA\_40W  
BB\_Upper = SMA\_40W + (STD\_40W × 2)  
BB\_Lower = SMA\_40W - (STD\_40W × 2)  
<br>
IF Current\_Price <= BB\_Lower → 8점  
IF Current\_Price <= BB\_Lower + (BB\_Mid - BB\_Lower) × 0.33 → 5점  
IF Current\_Price < BB\_Mid → 2점  
IF Current\_Price >= BB\_Mid → 0점  
<br>
최강 매수 신호: RSI\_14W <= 30 AND Current\_Price <= BB\_Lower  
\`\`\`

\## 4. 가중치 기반 최종 판단 로직  
<br>
| 카테고리 | 요소 | 만점 |  
|---|---|---|  
| C1 가치 평가 | F1 + F2 | 30점 |  
| C2 기업 퍼포먼스 | F3 + F4 + F5 | 30점 |  
| C3 시장 심리 | F6 + F7 | 20점 |  
| C4 매수 타이밍 | F8 + F9 + F10 | 20점 |  
| \*\*합계\*\* | | \*\*100점\*\* |  
<br>
\`\`\`  
Total\_Score = F1 + F2 + F3 + F4 + F5 + F6 + F7 + F8 + F9 + F10  
\`\`\`  
<br>
| 총점 | 신호 | 행동 지침 |  
|---|---|---|  
| 80점 이상 | Strong Buy | 즉시 1차 매수 집행 |  
| 60 ~ 79점 | Watch & Accumulate | 소량 분할 매수 시작 |  
| 60점 미만 | Hold / Pass | 관망 |  
<br>
\*\*판단 원칙\*\*  
\- C2에서 F3 또는 F4가 0점이면 총점과 무관하게 매수 신호 보류 → \`C2\_BLOCKED\` 상태로 처리  
\- F9(RSI)와 F10(볼린저밴드)은 반드시 동시 해석  
\- 업데이트 주기: 매주 금요일 장 마감 후 주봉 종가 기준

\## 5. score\_cache DB 컬럼 정의 (201-05 반영)  
<br>
C2 필터 상태를 DB와 UI까지 연결하기 위해 아래 컬럼을 score\_cache 테이블에 추가한다.  
<br>
\`\`\`sql  
\-- score\_cache 추가 컬럼  
signal\_201  TEXT     -- 'STRONG\_BUY' / 'WATCH' / 'HOLD' / 'C2\_BLOCKED'  
c2\_blocked  BOOLEAN  -- F3=0 OR F4=0 시 TRUE  
<br>
\-- 신호 판정 로직  
IF F3 == 0 OR F4 == 0:  
    signal\_201 = 'C2\_BLOCKED'  -- 총점과 무관하게 보류  
ELIF Total\_Score >= 80:  
    signal\_201 = 'STRONG\_BUY'  
ELIF Total\_Score >= 60:  
    signal\_201 = 'WATCH'  
ELSE:  
    signal\_201 = 'HOLD'  
\`\`\`  
<br>
\*\*UI 표시 기준\*\*  
\- 총점 80점 + \`C2\_BLOCKED\` → 점수는 높지만 \`⚠️ C2 이익 체력 0점 — 매수 보류\` 배지 표시  
\- 대시보드 필터에 \`C2\_BLOCKED\` 항목 별도 분류
