## 203\. 배당 지속가능성 판단

\# 배당 지속가능성 판단 로직 v1.1  
<br>
\---  
<br>
\## 1. 목표  
매수 시점이 확인된 종목에 대해, 배당이 미래에도 유지되고 성장할 수 있는지를 \*\*과거 재무제표(정량 검증)\*\* 와 \*\*현재 뉴스·선행지표(정성 검증)\*\* 두 축으로 판단한다.  
<br>
\- 배당 컷(Dividend Cut) 또는 배당 동결의 징후를 사전에 포착  
\- 단일 지표 맹신 금지. 3대 재무제표(현금흐름표, 손익계산서, 재무상태표)를 유기적으로 교차 검증  
\- 이상 징후 감지 시 → 비중 축소 또는 전량 매각 의사결정의 근거로 활용  
\- \*\*판단 주기\*\*: 과거 재무 검증은 분기 실적 발표 후 업데이트 / 뉴스·선행지표는 상시 모니터링  
<br>

\## 2. 다루는 요소 정의  
<br>
\### LAYER 1 — 과거 재무제표 기반 정량 검증 (70점)  
과거 10년치 데이터를 기준으로 기업의 배당 지급 체력을 검증한다. 10년이라는 기간은 최소 1회 이상의 거시경제 위기를 포함하므로, 실질적인 스트레스 테스트와 동일하다.  
<br>
\*\*현금흐름표 — "배당의 실질적 원천"\*\*  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| L1-F1 | 잉여현금흐름(FCF) 커버리지 | FCF가 총 배당금 지급액을 초과하는지. 배당은 장부 이익이 아닌 실제 현금에서 지급됨. FCF 적자 상태의 배당은 차입이나 자산 매각에 의존하는 것으로 지속 불가. 1년 Fail → Yellow Flag / 2년 연속 Fail → Red Flag |  
| L1-F2 | 현금배당성향 (Cash Payout Ratio) | 총 배당금 ÷ FCF. 순이익 기준 배당성향보다 더 정직한 안전성 지표. 70% 미만이면 위기 시에도 배당 유지 가능한 안전마진 |  
<br>
\*\*손익계산서 — "이익 체력과 이자 부담"\*\*  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| L1-F3 | EPS 성장률 (장기) | 배당금의 장기 우상향은 결국 이익 성장에서 나옴. 이익 성장 없이 배당성향만 높여 배당금을 늘리는 것은 미래 갉아먹기 |  
| L1-F4 | 이자보상배율 | 영업이익 ÷ 이자비용. 채권자에게 이자를 갚고도 주주에게 배당을 줄 여력이 있는지. 3 미만으로 하락 시 경기 침체기 배당 삭감 위험 급등 |  
<br>
\*\*재무상태표 — "위기 대응 방어력"\*\*  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| L1-F5 | 이익잉여금 추이 | 과거부터 누적된 실질 배당 재원. 매년 증가 추세인지 확인. 누적 결손금 상태는 배당 지속 불가 경고 신호 |  
| L1-F6 | 부채비율 (D/E Ratio) | 과도한 레버리지는 경기 침체 시 현금흐름을 경직시켜 배당 삭감의 1순위 원인. 업종 평균 D/E는 tickers.json에 수동 입력 고정값으로 관리. 연 1회 갱신 |  
| L1-F7 | 유통주식수 변화 추이 | 자사주 매입으로 유통주식수가 감소하면 총 배당 지급액을 늘리지 않아도 주당 배당금(DPS)이 자연 상승. 숨은 배당 성장 동력 |  
<br>
\*\*장기 성장 동기화 검증\*\*  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| L1-F8 | ROIC vs WACC | 투하자본수익률이 자본비용을 지속적으로 상회해야 초과 이익이 쌓이고 배당으로 환원 가능. 10년 평균 기준 비교. WACC는 valuation\_bands.json에 수동 입력 고정값으로 저장, 연 1회 갱신 |  
| L1-F9 | DGR vs FCF 성장률 동기화 | 배당성장률이 FCF 성장률을 지속적으로 초과하면 조만간 배당 인상이 멈추거나 삭감될 가능성. 핵심 경고 지표 |

\### LAYER 2 — 현재 뉴스 및 선행지표 기반 정성 검증 (30점)  
과거 재무제표는 최소 1분기 이상의 후행 데이터다. 현재 시점에서 배당 컷의 징후를 선제적으로 포착하기 위해 실시간 선행 데이터를 추가로 검증한다.  
<br>
\*\*시장 컨센서스 — "월가가 지금 기업을 어떻게 보는가"\*\*  
<br>
| 요소 ID | 요소명 | 설명 |  
|---|---|---|  
| L2-F1 | 애널리스트 EPS 추정치 방향 | 분기 실적 발표 전이라도 애널리스트들이 EPS·FCF 추정치를 일제히 하향 조정하기 시작하면 현금배당성향 안전마진이 단숨에 붕괴될 수 있음 |  
| L2-F2 | 내부자 거래 이상 여부 | CEO·CFO 등 핵심 경영진의 대량 자사주 매도는 단기 현금흐름 악화를 사전에 인지한 도피성 신호일 수 있음. SEC Form 4 기준 |  
<br>

| L2-F3 | 뉴스 톤(Tone) 및 키워드 분석 | Alpha Vantage NEWS\_SENTIMENT API 활용. 어닝콜 트랜스크립트 직접 수집 불가(FMP 무료 플랜 미지원)로 인해 실적 발표 전후 뉴스 감성으로 근사값 처리. 긍정 키워드(배당 인상, 자사주 매입 확대, 현금흐름 개선) / 부정 키워드(현금 보수적 관리, 자본 배치 재검토, 유동성 확보 우선) 감지. ※ 트랜스크립트 직접 분석 대비 정확도 낮아지는 트레이드오프 인지 필요 |  
| L2-F4 | 가이던스 변화 방향 | 경영진이 발표하는 미래 실적 전망의 상향/동결/하향 방향. 연속 하향은 다음 분기 FCF 감소 전조 |

\## 3. 요소별 판별 로직  
<br>
\### LAYER 1 — 재무제표 정량 로직  
<br>
\*\*L1-F1 — FCF 커버리지 (절대 관문)\*\*  
\`\`\`  
FCF\_Coverage = FCF / Total\_Dividends\_Paid  
<br>
\# 일시적 Fail vs 구조적 Fail 구분  
IF FCF\_Coverage >= 1.0 → Pass (정상)  
IF FCF\_Coverage < 1.0 (1년) → Yellow Flag 발동 (경고, 추가 매수 주의)  
IF FCF\_Coverage < 1.0 (2년 연속) → Red Flag 발동 → 이하 점수 무의미  
<br>
\# 5년 추세 반드시 확인  
\# Yellow Flag 상태에서 다음 분기 회복 시 자동 해제  
\# Red Flag는 2회 연속 갱신(2주) 기준 0개 유지 시 해제  
\`\`\`  
<br>
\*\*L1-F2 — 현금배당성향\*\*  
\`\`\`  
Cash\_Payout\_Ratio = Total\_Dividends\_Paid / FCF  
IF Cash\_Payout\_Ratio <= 0.70 → 10점  
IF Cash\_Payout\_Ratio <= 0.85 → 5점  
ELSE → 0점 + Red Flag  
※ 유틸리티·리츠 등 자본 집약적 업종은 기준값 0.85 이하로 완화 적용  
\`\`\`  
<br>
\*\*L1-F3 — EPS 성장률 (10년 CAGR)\*\*  
\`\`\`  
IF EPS\_10Y\_CAGR >= 0.07 → 10점  
IF EPS\_10Y\_CAGR >= 0.03 → 6점  
IF EPS\_10Y\_CAGR >= 0 → 3점  
ELSE → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L1-F4 — 이자보상배율\*\*  
\`\`\`  
Interest\_Coverage = Operating\_Income / Interest\_Expense  
IF Interest\_Coverage >= 5 → 10점  
IF Interest\_Coverage >= 3 → 6점  
IF Interest\_Coverage >= 1.5 → 2점  
ELSE → 0점 + Red Flag  
10년 추세 하락 여부 별도 체크  
\`\`\`  
<br>
\*\*L1-F5 — 이익잉여금 추이\*\*  
\`\`\`  
IF 이익잉여금 매년 증가 (10년 중 7년 이상) → 10점  
IF 이익잉여금 증감 혼재 → 5점  
IF 이익잉여금 감소 추세 또는 누적 결손금 → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L1-F6 — 부채비율\*\*  
\`\`\`  
\# 업종 평균 D/E: tickers.json에 수동 입력 고정값으로 관리  
\# 갱신 주기: 연 1회 (금리 환경 변화 시)  
\# Bootstrap 시 초기값 입력 필수  
<br>
비교 기준: tickers.json의 sector\_de\_ratio (업종 평균 D/E)  
IF D/E <= 업종 평균 × 0.7 → 10점  
IF D/E <= 업종 평균 × 1.0 → 6점  
IF D/E <= 업종 평균 × 1.3 → 3점  
ELSE → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L1-F7 — 유통주식수 변화 추이\*\*  
\`\`\`  
Shares\_Trend = (Shares\_Current - Shares\_10Y\_Ago) / Shares\_10Y\_Ago  
IF Shares\_Trend <= -0.10 → 5점  
IF Shares\_Trend <= 0 → 3점  
ELSE → 0점  
\`\`\`  
<br>
\*\*L1-F8 — ROIC vs WACC\*\*  
\`\`\`  
\# WACC: valuation\_bands.json에 수동 입력 고정값으로 저장  
\# 갱신 주기: 연 1회 (금리 환경이 크게 바뀔 때만 수정)  
\# Bootstrap 시 초기값 입력 필수 항목  
<br>
IF ROIC\_10Y\_Avg > WACC + 0.03 → 5점  
IF ROIC\_10Y\_Avg > WACC → 3점  
ELSE → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L1-F9 — DGR vs FCF 성장률 동기화\*\*  
\`\`\`  
IF DGR\_5Y <= FCF\_Growth\_5Y → 10점  
IF DGR\_5Y <= FCF\_Growth\_5Y + 0.03 → 5점  
ELSE → 0점 + Red Flag  
※ 3년 연속 Fail이면 배당 동결 또는 삭감 고위험 경고  
\`\`\`

\### LAYER 2 — 선행지표 정성 로직  
<br>
\*\*L2-F1 — 애널리스트 EPS 추정치 방향\*\*  
\`\`\`  
기준: 최근 4주간 다음 분기 EPS 컨센서스 변화 방향  
IF 상향 조정 → 10점  
IF 유지 → 5점  
IF 하향 조정 → 0점 + Yellow Flag  
IF 대폭 하향 (-10% 이상) → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L2-F2 — 내부자 거래 이상 여부\*\*  
\`\`\`  
기준: 최근 90일 SEC Form 4 (FMP insider\_transactions)  
IF 내부자 순매수 → 5점  
IF 내부자 거래 없음 → 3점  
IF 소규모 매도 → 1점  
IF 대규모 매도 (보유 지분 10% 이상) → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L2-F3 — 뉴스 톤 및 키워드 분석\*\*  
\`\`\`  
\# 데이터 소스: Alpha Vantage NEWS\_SENTIMENT API  
\# 어닝콜 트랜스크립트 직접 수집 불가 → 실적 발표 전후 뉴스 감성으로 근사값 처리  
<br>
긍정 키워드: 배당 인상, 자사주 매입 확대, 가이던스 상향, 현금흐름 개선  
부정 키워드: 현금 보수적 관리, 자본 배치 재검토, 유동성 확보 우선, 비용 절감 집중  
<br>
IF 긍정 키워드 우세, 부정 키워드 없음 → 10점  
IF 혼재 → 5점  
IF 부정 키워드 감지 → 0점 + Yellow Flag  
IF 배당 정책 재검토 직접 언급 → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L2-F4 — 가이던스 변화 방향\*\*  
\`\`\`  
IF 가이던스 상향 (Raise) → 5점  
IF 가이던스 유지 (Maintain) → 3점  
IF 가이던스 하향 (Lower) → 0점 + Yellow Flag  
IF 가이던스 철회 (Withdraw) → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L2-F5 — 신용등급 변동 뉴스\*\*  
\`\`\`  
IF 등급 유지 또는 상향 → 5점  
IF 전망 부정적(Negative) 전환 → 2점 + Yellow Flag  
IF 등급 강등 (Downgrade) → 0점 + Red Flag  
\`\`\`  
<br>
\*\*L2-F6 — 규제·정책 리스크 뉴스\*\*  
\`\`\`  
기준: 최근 90일 뉴스 키워드 스크리닝 (Alpha Vantage NEWS\_SENTIMENT)  
IF 주요 규제 리스크 없음 → 5점  
IF 규제 논의 중 (확정 아님) → 2점 + Yellow Flag  
IF 규제 확정 또는 직접 영향 판결 → 0점 + Red Flag  
\`\`\`

\### 총점 계산

// 1단계: L1-F1 절대 관문 확인

\## 6. 시뮬레이션 기준값 — DGR 선형 가중평균  
<br>
배당 지속가능성이 \*\*STRONG\*\* 등급으로 확인된 종목에 한해 시뮬레이션을 진행한다.  
<br>
\### 기준값 산출 방식  
<br>
| 연도 | 가중치 | 비율 |  
|---|---|---|  
| Y-4 (5년 전) | 1 | 6.7% |  
| Y-3 (4년 전) | 2 | 13.3% |  
| Y-2 (3년 전) | 3 | 20.0% |  
| Y-1 (2년 전) | 4 | 26.7% |  
| Y-0 (최근 1년) | 5 | 33.3% |  
| \*\*합계\*\* | \*\*15\*\* | \*\*100%\*\* |  
<br>
\`\`\`  
Weighted\_DGR = (DGR\_Y4×1 + DGR\_Y3×2 + DGR\_Y2×3 + DGR\_Y1×4 + DGR\_Y0×5) / 15  
\`\`\`  
<br>
최근 2년(Y-1, Y-0)이 전체의 60% 비중을 차지하여 최근 배당 성장 추세 변화가 기준값에 즉각 반영된다.  
<br>
\### 시나리오별 DGR 적용  
<br>
| 시나리오 | DGR 적용값 | 가정 |  
|---|---|---|  
| Base Case | Weighted\_DGR | 현재 추세 그대로 유지 |  
| Bull Case | Weighted\_DGR + 2%p | 실적 개선, 배당 인상 가속 |  
| Bear Case | max(Weighted\_DGR - 3%p, -5%) | 성장 둔화 또는 1회 배당 동결. 최대 -5%까지 허용 |  
| Freeze | 0% | Red Flag 발동 시 강제 적용 |  
<br>
\### 시뮬레이션 출력 항목 (10년 기준)  
\- 연도별 예상 주당 배당금(DPS)  
\- 매수 시점 기준 누적 수령 배당금 총액  
\- 시나리오별 예상 배당수익률 (매수가 기준)  
\- Bear Case 기준 손익분기 회복 기간
